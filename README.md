# Day-1
Build a responsive Scientific Calculator with a modern UI. Support basic arithmetic, square root, power, percentage, factorial, logarithm, sine, cosine, and tangent. Include Clear, Backspace, decimal support, error handling, keyboard support, calculation history, dark/light mode, and create it using HTML, CSS, and JavaScript.

## **Scientific Calculator (Day-01)**

### **Project Description**

The **Scientific Calculator** is a responsive web application built using **HTML, CSS, and JavaScript**. It provides both basic and advanced mathematical operations through a clean, modern, and user-friendly interface. The calculator works smoothly on desktop and mobile devices, making it suitable for students, teachers, and professionals.

---

## **Features**

* ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* √ Square root calculation
* xʸ Power (Exponent) calculation
* % Percentage calculation
* ! Factorial calculation
* log Logarithm function
* sin, cos, tan trigonometric functions
* 🔢 Decimal number support
* ⌫ Backspace to remove the last digit
* 🗑️ Clear button to reset calculations
* ⚠️ Error handling for invalid expressions
* ⌨️ Keyboard support for faster input
* 📝 Calculation history
* 🌙 Dark and Light mode
* 📱 Responsive design for all screen sizes
